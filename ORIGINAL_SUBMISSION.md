## Idea
Stray animals and livestock frequently ingest hazardous plastic waste from municipal dumps and streets, leading to fatal health complications and ecological damage. My solution is an automated, hardware-based IoT detection system designed for localized deployment. Utilizing an embedded microcontroller like a Raspberry Pi Pico paired with a low-power edge-AI vision model, the device will visually distinguish plastic waste in high-risk zones. Upon detection, the system will trigger real-time alerts to a centralized cloud dashboard via cellular or LoRa network for municipal authorities. This low-cost, scalable hardware approach directly reduces animal mortality, provides actionable data for targeted cleanups, and optimizes city waste management operations.
